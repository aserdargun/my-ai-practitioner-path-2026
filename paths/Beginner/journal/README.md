# Learning Journal

This folder contains your reflections and learnings throughout your AI practitioner journey.

## Purpose

- Process what you've learned
- Track challenges and solutions
- Build a record of growth
- Develop meta-learning skills

## Structure

### Weekly Reflections

Create a file for each week:

```
journal/
├── week-01.md
├── week-02.md
├── week-03.md
└── ...
```

### Monthly Retrospectives

At the end of each month, create:

```
journal/
├── month-01-retro.md
├── month-02-retro.md
└── ...
```

## Weekly Reflection Template

Copy this template for each week:

```markdown
# Week [X] Reflection

**Date**: [Date]
**Month**: [Month topic]

## What I Learned

- [Learning 1]
- [Learning 2]

## What I Built

- [Project/task completed]

## Challenges

- [Challenge 1]: [How I solved it]
- [Challenge 2]: [How I solved it]

## Questions

- [Question I still have]

## Next Week

- [Goal for next week]
```

## Monthly Retrospective Template

```markdown
# Month [X] Retrospective

**Topic**: [Month topic]
**Dates**: [Start] - [End]

## Accomplishments

- [What I completed]

## Key Learnings

1. [Most important thing learned]
2. [Second most important]
3. [Third most important]

## Challenges Overcome

- [Challenge]: [Solution]

## What Would I Do Differently?

- [Reflection]

## Favorite Moment

[Best part of the month]

## Goals for Next Month

- [Goal 1]
- [Goal 2]
```

## Tips

1. **Write regularly** - Don't wait until the end
2. **Be honest** - Include struggles
3. **Be specific** - Details are valuable
4. **Connect concepts** - Link new to old
5. **Review periodically** - Reread past entries

## Using /retro

Run `/retro` at the end of each week for guided reflection prompts.

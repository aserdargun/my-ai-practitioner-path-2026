# Month 12: Capstone Project

**Duration**: 4 weeks
**Focus**: End-to-end project integrating all skills

---

## Objectives

By the end of this month, you will:

- [ ] Plan and scope a complete project
- [ ] Apply skills from all previous months
- [ ] Build an end-to-end application
- [ ] Document and present your work
- [ ] Reflect on your learning journey

---

## Weekly Breakdown

| Week | Focus | Deliverables |
|------|-------|--------------|
| Week 1 | Planning & Design | Project plan |
| Week 2 | Core Development | Working prototype |
| Week 3 | Integration & Testing | Complete application |
| Week 4 | Documentation & Demo | Published project |

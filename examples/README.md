# Examples

This folder contains example projects and code snippets demonstrating concepts from the curriculum.

## Structure

```
examples/
├── python-basics/        # Month 01-02 examples
├── data-analysis/        # Month 03-04 examples
├── web-apis/             # Month 06 examples
└── ml-projects/          # Month 07-11 examples
```

## Using Examples

Each example includes:
- Working code
- Comments explaining concepts
- Instructions to run
- Exercises to try

## Contributing Examples

When you complete a project, consider adding a simplified version here to help future learners!

### Guidelines

1. Keep examples focused on one concept
2. Include clear comments
3. Add a README with instructions
4. Test before committing
